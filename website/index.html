The Agent Gateway

Let AI agents draft messages to your contacts — but you decide what actually gets sent.

The Gateway is a human-in-the-loop approval system that prevents AI from impersonating you. Agents queue messages via API, and you review every single one before it goes out.

Perfect for automation workflows, AI assistants, and anyone who wants AI help without giving up control.

1

Agent Queues Message

AI agents submit email, SMS, or iMessage drafts via secure HTTP API

2

Review in contactCMD

Messages appear in your approval queue with full context

3

Approve or Deny

One keystroke to approve, edit, or reject each message

4

Sent on Your Behalf

Only approved messages reach your contacts
